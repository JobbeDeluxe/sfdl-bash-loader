# SFDL Bash Loader

Zum einfachen herunterladen mit SFDL Dateien unter Linux und Mac

## Installation
Auspacken: unzip sfdl-bash-loader-X.X.zip
Ins Verzeichnis wechseln: cd sfdl-bash-loader-X.X/sfdl_bash_loader  
Download starten: ./start.sh
