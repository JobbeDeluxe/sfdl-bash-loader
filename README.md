# SFDL Bash Loader

Zum einfachen herunterladen mit SFDL Dateien unter Linux und Mac

## Installation
Auspacken: unp sfdl_bash_loader_vX.X.zip  
Ins Verzeichnis wechseln: cd sfdl_bash_loader  
Download starten: ./start.sh
